# 夏季生物

![Software License](https://img.shields.io/badge/license-MIT-brightgreen.svg?style=flat-square)

为群里增加夏季生物活动吧！

## 使用说明

`.夏季生物` // 查看当前状态

`.夏季生物 (开始|停止)` // 夏季生物开始/停止活动

`.夏季生物 (help|帮助)` // 查看帮助

`.夏季生物 释放 (蚊子|蟑螂)` // 手动向群里释放生物

当有对应生物活动时，可以使用如下命令攻击：

`#拍` // 攻击蚊子

`#踩` // 攻击蟑螂
